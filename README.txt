Installation:

Prerequisites:
- Node.js
- npm

Steps to run the project:

Install dependencies:
npm install

Start the server:
node server.js

Open your browser and go to http://localhost:3000 to see the application.

How to Use
On the main page, you can create new posts by filling out the form and clicking the Create Post button.
The created posts will appear below, where you can edit or delete them.

Folder Structure:

posts-app/
├── public/
│   ├── index.html      # Main HTML file
│   ├── style.css       # Styles for the app
│   └── script.js       # JavaScript for frontend logic
├── server.js           # Backend server logic
└── package.json        # Project dependencies
